# Sports Favorites App

Sports Favorites App is a simple and elegant SwiftUI application that allows users to view and explore a list of favorites in various sports. This project serves as a practical introduction to **SwiftUI**, showcasing its powerful and declarative UI development approach.

## Features

- **Favorites Listing**: Displays a list of favorites in sports categories.
- **Detail View**: Provides more information about each favorite item.
- **Navigation View Integration**: Smooth transitions between the list and detail screens.
- **SwiftUI Design**: Clean and responsive interface with a modern look.

## How It Works

1. **Model Creation**:
   - Built data models to represent the sports favorites and their associated details.

2. **Navigation View**:
   - Used a **Navigation View** to structure the app and enable seamless redirection to detail views.

3. **Detail View**:
   - Designed a detailed screen for each favorite item, displaying additional information.

4. **Small UI Adjustments**:
   - Finalized the app by fine-tuning minor details for an optimal user experience.

## Requirements

- Xcode 14+ 
- iOS 15+ 
- Swift 5.7+

## Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/your_username/sports-favorites-app.git
   cd sports-favorites-app
   ```

2. Open the project in Xcode:
   ```bash
   open SportsFavoritesApp.xcodeproj
   ```

3. Run the project on a simulator or a physical device.
